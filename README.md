# Recipes List

## To use locally
1. Clone this repo
2. Run `npm i` (wait for install to finish)
2. Run `npm start`
3. Go to `http://localhost:3000/` if it doesn't open automatically

### Test the production build
1. Make sure you have [serve](https://github.com/zeit/serve) installed
2. Run `npm run build`
3. Run `serve -s build`
